# API Reference

- [Algorithms](algorithms.md) — core solvers for discrete modulus: a
  direct LP solver (`matrix_modulus`) and the incremental "basic
  algorithm" (`modulus`).
- Families — functor classes/operators implementing families of objects
  (`ShortestObjectFinder`s) for use with `algorithms.modulus`:
  [NetworkX families](families/networkx_families.md),
  [family operators](families/family_operators.md).
- [Demo](demo.md) — small graphs and brute-force helpers for
  demonstrations.
- [Protocols](protocols.md) — shared result types and structural
  interfaces.
