Detailed Description

The reusable max-flow network for cunningham_min / graph_vulnerability / spanning_tree_modulus, built once per graph by create_flow_graph.

Bundles the flow graph together with handles to the specific edges whose capacities get updated on every call (looked up once here, since FlowGraph has parallel edges between some vertex pairs – see create_flow_graph's implementation comments).
